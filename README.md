# FormulaireTravailCSS
Dans le cadre de ma formation en tant que stagiaire, un exercice a été réalisé sur Visual Studio Code le 12 mai 2026.

L’objectif de cet exercice était de modifier l’apparence graphique d’un site web sans toucher au fichier HTML.
Toutes les modifications ont donc été effectuées uniquement à l’aide du CSS afin de travailler la mise en page, les couleurs, les polices, les espacements et l’amélioration visuelle générale du site.
